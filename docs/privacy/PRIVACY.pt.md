# Política de Privacidade

_Última atualização: 22 de abril de 2026_

O openings.dev foi criado para facilitar a descoberta de vagas de tecnologia publicadas por comunidades. Esta política explica quais dados tratamos e com qual finalidade.

## 1. Dados processados

Tratamos principalmente dados públicos de vagas publicados em repositórios do GitHub.

Para operação da plataforma, também podemos processar metadados técnicos limitados, como logs, telemetria de erro e sinais agregados de uso.

## 2. Uso dos dados

Usamos dados para indexar vagas, melhorar busca e filtros, e manter segurança e estabilidade do serviço.

Não vendemos dados pessoais.
