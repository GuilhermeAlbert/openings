# Política de Privacidad

_Última actualización: 22 de abril de 2026_

openings.dev facilita el descubrimiento de vacantes publicadas por comunidades. Esta política explica qué datos tratamos y con qué propósito.

## 1. Datos procesados

Procesamos principalmente datos públicos de vacantes publicados en repositorios de GitHub.

Para operar la plataforma, también podemos procesar metadatos técnicos limitados como logs, telemetría de errores y señales agregadas de uso.

## 2. Uso de datos

Usamos datos para indexar vacantes, mejorar búsqueda y filtros, y mantener seguridad y estabilidad del servicio.

No vendemos datos personales.
